# MyPortfolio - Personal Android Portfolio App 

An open-source Android portfolio app I developed to showcase my personal projects, the competitions I’ve entered, and my journey in software development.

## About the Project
This app serves as the digital CV of a 15-year-old developer driven by a vision for computer engineering. The app features:
* The **Umay** project, a P2P-architecture-based initiative that was a regional finalist in the TÜBİTAK 2209-A competition,
* My game development experiences (Data Platformer),
* My social responsibility and educational seminars.

## Technologies Used
The app has been developed in accordance with modern Android standards:
* **Programming Language:** Kotlin
* **User Interface (UI):** Jetpack Compose
* **Version Control:** Git & GitHub

## Application Features
* **About Me:** My career goals, educational background and vision for technology.
* **Skills:** Lua, Python, Java, C#, Rust and Android components.
* **My Projects:** Details of all the open-source and personal projects I have developed from the past to the present.
* **Contact Information:** Quick access to my LinkedIn, GitHub and itch.io accounts.

## Installation (To Run the Project on Your Own Computer)
1. Clone this repository to your computer:
   ```bash
   git clone https://github.com/DATA35DELETE/My-Portfolio.git
   ```

2. Open the project in Android Studio.

3. Wait for the necessary Gradle packages to be installed.

4. Connect your device or launch an emulator and press the “Run” button.
